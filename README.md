# FruitBat-The-Game
Must have Scratch Editor to play from download

Playable from Scratch Website @ https://scratch.mit.edu/projects/1288733398

Made for CS50. In this game, you play as a hungry fruit bat trying to collect as much fruit within 1 minute as possible. 

Press the green flag after opening the game, then press SPACE to START. 

Use the ARROW keys to move. 

Each fruit has a different point value, Strawberries: 1 Cherries: 2 Pears: 3 Bananas: 4
